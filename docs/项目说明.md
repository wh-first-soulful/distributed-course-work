# 项目说明

## 项目名称
LightMall 用户服务

## 项目目标
实现一个基于Spring Boot的用户服务，提供用户注册、登录和获取用户信息的功能。

## 技术栈

### 后端框架
- Spring Boot 2.7.15
- MyBatis Plus 3.5.3.1
- JWT 0.11.5
- MySQL 8.0.33
- Knife4j 3.0.3 (API文档工具)

## 主要功能

### 用户注册
- 接收用户注册信息
- 检查用户名是否已存在
- 创建新用户

### 用户登录
- 验证用户名和密码
- 生成JWT token
- 返回用户信息和token

### 获取用户信息
- 根据用户ID获取用户信息

## API接口

### 用户注册
- 路径：`/user/register`
- 方法：POST
- 请求体：
  ```json
  {
    "username": "用户名",
    "password": "密码",
    "nickname": "昵称",
    "email": "邮箱",
    "phone": "手机号"
  }
  ```
- 响应：
  ```json
  {
    "code": 200,
    "message": "success",
    "data": {
      "id": 1,
      "username": "用户名",
      "nickname": "昵称",
      "email": "邮箱",
      "phone": "手机号",
      "status": 1,
      "createTime": "2026-03-18 09:00:00",
      "updateTime": "2026-03-18 09:00:00"
    }
  }
  ```

### 用户登录
- 路径：`/user/login`
- 方法：POST
- 请求体：
  ```json
  {
    "username": "用户名",
    "password": "密码"
  }
  ```
- 响应：
  ```json
  {
    "code": 200,
    "message": "success",
    "data": {
      "token": "JWT token",
      "user": {
        "id": 1,
        "username": "用户名",
        "nickname": "昵称",
        "email": "邮箱",
        "phone": "手机号",
        "status": 1,
        "createTime": "2026-03-18 09:00:00",
        "updateTime": "2026-03-18 09:00:00"
      }
    }
  }
  ```

### 获取用户信息
- 路径：`/user/get/{id}`
- 方法：GET
- 响应：
  ```json
  {
    "code": 200,
    "message": "success",
    "data": {
      "id": 1,
      "username": "用户名",
      "nickname": "昵称",
      "email": "邮箱",
      "phone": "手机号",
      "status": 1,
      "createTime": "2026-03-18 09:00:00",
      "updateTime": "2026-03-18 09:00:00"
    }
  }
  ```

## 运行环境
- JDK 11+
- MySQL 8.0+
- Maven 3.6+

## 运行步骤
1. 执行 `init.sql` 创建数据库和表
2. 修改 `application.yml` 中的数据库配置
3. 执行 `mvn clean install` 构建项目
4. 执行 `mvn spring-boot:run` 启动应用
5. 访问 `http://localhost:9081/doc.html` 查看API文档
